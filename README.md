# BMAD Interactive Page

Interactive French learning page explaining BMAD, Convoque, and agentic product lifecycle steering from the June 30, 2026 lecture screenshots.

[![Deploy to Netlify](https://www.netlify.com/img/deploy/button.svg)](https://app.netlify.com/start/deploy?repository=https://github.com/Ganda15/bmad-interactive)

## What This Page Explains

- How BMAD structures agentic software delivery.
- Where BMAD sits in the AI-driven / spec-driven / intent-driven development landscape.
- Why Convoque extends BMAD toward discovery, readiness, operations, knowledge, governance, and adoption.
- Why the product lifecycle is better understood as a gravity model than a linear pipeline.
- Why knowledge engineering, entropy monitoring, domain mesh maturity, and governance matter when using AI agents at scale.

## Contents

- `index.html` - the complete interactive page with embedded CSS and JavaScript.
- `assets/` - the 18 lecture screenshots used as visual sources.

## Interactive Features

- Concept tabs for BMAD principles, skills, artifacts, agents, orchestration, and Document as Code.
- Phase explorer for the hybrid BMAD / Convoque product workflow.
- Gravity model explorer.
- Knowledge engineering flow.
- Screenshot gallery with lightbox.
- Mini quiz.
- Oral-ready summary section.

## Run Locally

Open `index.html` directly in a browser.

No build step is required.

## Deployment

This is a static site. It can be deployed directly to Netlify by publishing the repository root.

Recommended Netlify settings:

- Build command: empty
- Publish directory: `.`

## Source

The content is based on lecture screenshots captured on June 30, 2026 and organized into a pedagogical interactive page.
